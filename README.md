## Requirements

- Netlify account
- Contentful account
- yarn v1.x

## Install dependencies

```bash
yarn
```

## Create `/.env` file

```text
SPACE=...
ENV=...
ACCESS_TOKEN=...
```

## Build

```bash
yarn local-build
```

## Lauch dev server

```bash
yarn local-serve
```
